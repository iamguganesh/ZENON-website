# 🌌 NOVAZONE ARCHIVE

A sci-fi universe archive website featuring characters, ships, and worlds from the NOVAZONE multiverse.

---

## 🚀 About the Project

NOVAZONE ARCHIVE is a visual prototype of a fictional universe containing:
- ZENON fictional universe
- Future expansion zones (coming soon)

It is designed as an interactive character database.

---

## 👾 Features

- Character cards system
- Flip card (front: image / back: details)
- Hold-to-zoom image inspection
- Universe filtering system
- Responsive grid layout (desktop + mobile)
